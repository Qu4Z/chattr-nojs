Welcome to SITE

This is a barebones web chat client, with a focus on "minimum viable" design. Messages are interpreted as html, and you're free to do what you like.

Conversations take place in rooms. Roooms store the last few messages for up to 3 days of inactivity, after which the room is removed/regenerated.

Simply go to the room you'd like in the url bar, like so: SITE/all, SITE/room, SITE/...
